Learn to Build a Ruby Gem.

A FREE 10-day email course helping you build a Ruby Gem from start to finish.

What you'll get out of this FREE course:

  • 5 lessons delivered via email over 10 days
  • Step-by-step instructions on creating a Ruby gem
  • The steps necessary to contribute to other Open Source projects
  • Modular code to use in multiple Ruby applications
  • Increased familiarity with Bundler

About me:

My name is Brandon Hilkert and I've been feverishly building Ruby applications since 2009.

I first got started with open source Ruby gems by contributing to Sidekiq. Since then I've written about contributing to Open Source software and made a few fairly popular Ruby gems myself.

When I attempted to write my first gem, I was pretty confused about the standards and conventions necessary to integrate with existing Ruby tooling. So I wrote this course to in attempts to bridge that gap. I hope you find it useful.

Want to learn more?

The Build a Ruby Gem book covers additional topics like Rails integration and common configuration patterns. Multiple packages are available including 14 chapters of step-by-step code along with over 2+ hours of screencasts detailing my workflow working through each chapter.
